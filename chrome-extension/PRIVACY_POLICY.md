# Privacy Policy for LearnUs Video Downloader

**Last Updated:** [Date]

## Introduction

LearnUs Video Downloader is a Chrome extension for downloading lecture videos from Yonsei University's LearnUs platform.

## Data Collection and Storage

### What We Collect

1. **Username (Yonsei ID)**: Stored locally in browser to pre-fill login form.

2. **Password**: Not stored. Required for each login.

3. **Download Settings**: Stored locally in browser.

### How We Store Data

- Data stored locally using Chrome's `chrome.storage.local` API
- Data stored only on device, never transmitted to external servers
- Data not encrypted (plaintext in Chrome's local storage)

### Data Transmission

- **Login Credentials**: Transmitted directly to LearnUs servers. Not intercepted, stored, or transmitted to third-party servers.

- **Video Downloads**: Downloaded directly from LearnUs servers. Not hosted, stored, or transmitted by extension.

- **No Analytics**: No analytics, usage statistics, or telemetry collected.

## Third-Party Services

### LearnUs Platform
- Communicates with LearnUs platform (`https://ys.learnus.org`) for authentication and video access
- Communication required for extension functionality
- LearnUs privacy practices not controlled by extension

### Optional: Local Python Server
- Some features require local Python server
- Data processed locally
- No data sent to external servers

## Data Security

- Passwords never stored
- All data stored locally on device
- No cloud sync

## User Rights

User rights:
- Access: View stored data via Chrome extension storage API or options page
- Delete: Clear all stored data via "Clear Settings" button
- Control: Disable or uninstall extension at any time

## Children's Privacy

Intended for Yonsei University students and faculty. Does not knowingly collect data from children under 13.
